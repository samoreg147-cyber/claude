# 🗳️ Голосование 2026 — мобилизация в России осенью

Современный одностраничный лендинг для **независимого анонимного опроса общественного мнения**:
*«Как вы относитесь к возможной новой волне мобилизации в РФ осенью 2026 года?»*

🌐 **Живая версия:** https://samoreg147-cyber.github.io/claude/

## Возможности

- Три варианта ответа: *Не поддерживаю · Поддерживаю · Затрудняюсь ответить*
- Мгновенные результаты с анимированной диаграммой
- Полностью анонимно — ни форм, ни трекинга, ни cookies
- Возможность изменить свой голос
- Адаптивный дизайн, тёмная тема, поддержка `prefers-reduced-motion`
- Кнопка «Поделиться» (Web Share API + копирование ссылки)

## Технологии

Чистый HTML + CSS + JavaScript, без зависимостей. Разворачивается на **GitHub Pages**
через GitHub Actions (`.github/workflows/deploy.yml`) — Pages включается автоматически.

## Как считаются голоса

В статической версии голос сохраняется анонимно в `localStorage` браузера, поэтому счётчик
отражает данные вашего устройства. Чтобы вести **общий подсчёт** для всех посетителей,
задайте `CONFIG.apiBase` в `script.js` (URL бесплатного serverless-счётчика, например
Cloudflare Worker или Deno Deploy с эндпоинтами `GET /tally` и `POST /vote`). Клиентский
код уже готов к такому подключению.
