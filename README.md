# 🗳️ Голосование 2026 — опрос о мобилизации · инициатор «Новые люди»

Современный адаптивный лендинг для **независимого общественного опроса**, инициированного
партией **«Новые люди»**: *«Как вы относитесь к возможной новой волне мобилизации в РФ осенью 2026 года?»*

🌐 **Живая версия:** https://samoreg147-cyber.github.io/claude/

## Возможности

- Адаптивный дизайн под все устройства (телефон / планшет / десктоп), тёмная тема, анимации
- Фирменный бирюзовый цвет партии «Новые люди», логотип (`assets/novye-lyudi.svg`)
- Блок результатов голосования с анимированными полосами
- Раздел «Как принять участие», «Об инициаторе», FAQ и дисклеймер
- Кнопки входа/регистрации ведут на внешнюю платформу голосования
- Поддержка `prefers-reduced-motion`

## Настройка ссылки на голосование

Логика голосования реализуется на стороне организатора. Чтобы кнопки
«Войти и голосовать» / «Зарегистрироваться» вели на нужный адрес, откройте
`script.js` и укажите ссылку в одной строке:

```js
const CONFIG = {
  registrationUrl: "https://ваша-ссылка-на-опрос",
  openInNewTab: true,
};
```

Пока `registrationUrl` пустой, кнопки показывают уведомление вместо перехода.

## Технологии

Чистый HTML + CSS + JavaScript, без зависимостей. Хостинг — **GitHub Pages** (ветка `main`).

## Дисклеймер

Проект носит информационно‑просветительский характер — это неофициальный опрос
общественного мнения, инициированный партией «Новые люди». Он не является референдумом
или голосованием органов власти и не имеет юридической силы. Участие добровольное и анонимное.
